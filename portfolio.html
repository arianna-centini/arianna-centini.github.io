Progetto 1 - Counter

Un'applicazione JavaScript che permette di aumentare, diminuire e resettare il valore del counter.

Questo progetto è stato pubblicato sulla piattaforma CodePen il 31 maggio 2022. L'applicazione è stata sviluppata utilizzando HTML, CSS e JavaScript.

La demo del progetto è pubblica e si trova su CodePen (qui). Il codice sorgente dell'appplicazione è contenuto in una repository di GitHub (qui).
